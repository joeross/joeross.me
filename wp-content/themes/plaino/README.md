plaino
===

A powerful blogging theme with minimal layout. It has two widget areas and a full width page template along with the normal sidebar template.

#Features:

##Widget areas:
1. Right sidebar, max content width 300px.
2. Footer, max content width 320px.

##Customizer
1. Header background color
2. Header text color
3. Logo/Avatar
4. Optional Header image of 1280px x 300px

##Post Formats:
Aside and Standard.

##Featured images
Post and Pages can have featured images.

##Social Icons
Supports social icons menu.
