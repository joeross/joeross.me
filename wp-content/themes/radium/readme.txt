Radium is a clean, responsive and beautifully crafted WordPress theme for bloggers. Designed with the reader in mind, it features great typography and includes some of the best open-source web fonts. When it comes to WordPress themes, less is definitely more, so with Radium you get just the right amount of options and a clean design, letting your awesome content do the talking. It features custom header images and logo, custom link colors, custom fonts and the unique ability to set a post's featured image as header image. All these options are effortlessly handled via the Theme Customizer, allowing admins to live preview their changes. Radium also offers support for the following post formats: aside, image, video, quote, link. 

Radium is based on Undescores, a starter theme from Automattic: http://underscores.me
Underscores WordPress theme is licensed under GPLv2 or later.

Here’s a summary of what Radium has to offer:

* Responsive, mobile-first design
* Post formats support
* Standard compliant & localization ready
* Custom header image and logo
* Featured image as header background
* Custom typography
* Custom colors
* Widgetized footer
